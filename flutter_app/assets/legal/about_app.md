# ACERCA DE LA APLICACIÓN BEDIC

**BEDIC** es una plataforma geolocalizada, bilingüe y colaborativa, diseñada para conectar a los usuarios con lugares cercanos y permitirles compartir información verificada sobre ellos.

| Característica | Detalle |
| :--- | :--- |
| **Nombre** | Busca, Encuentra, Disfruta, Interactúa y Comparte |
| **Plataforma** | Android (Flutter) |
| **Backend** | Node.js / Express / MongoDB Atlas |
| **APIs de Datos** | Google Places, OpenStreetMap |
| **Colaboración**| Reportes ciudadanos con moderación y votos de verificación. |
| **Internacionalización**| Español e Inglés (i18n) |
